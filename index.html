C/C++ Build Tools Guide

Project Pitch

What are we making?

Programmers who want to learn more about C/C++ build tools will be disappointed by the lack of easy-to-use online resources. Existing material is too technical, unclear, or incomplete. To solve this problem, we propose the creation of a user-friendly C/C++ Build Tools Guide.

What does this mean?

Build tools include compilers, linkers, assemblers, and other automation tools. Together, these convert source code into a deliverable package, which may be an application for end users or a software module for other programmers. Our C/C++ Build Tools Guide will include instructions for choosing and using compilers, build systems, and other tools appropriately and effectively.

What's our plan?

Our team consists of six people with different programming backgrounds and experiences; we will incorporate both depth and generalizability for the various needs of our audience. Over the next two weeks, we will research build tools, organize information, and revise content for maximum comprehensibility.
